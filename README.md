# Pawtal Frontend

## Chạy project

```bash
cp .env.example .env
npm install
npm run dev
```

## Ghi chú

- Ảnh marketing lấy từ Cloudinary qua biến môi trường.
- Ảnh động như thú cưng, sản phẩm, dịch vụ lấy từ API backend.
- Nếu backend chưa có dữ liệu ảnh, giao diện sẽ tự fallback sang placeholder.
